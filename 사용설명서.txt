Sora2 Auto 사용설명서 (감성 친절 모드)

1. 설치
   - `1_처음설치.bat` 을 더블클릭합니다.
   - 인터넷이 연결되어 있어야 하며, 자동으로 pip와 필수 라이브러리( selenium, webdriver-manager )가 설치됩니다.

2. 프롬프트 작성
   - `prompts.txt` 파일을 열어 원하는 요청을 작성합니다.
   - 여러 개라면 `|||` 로 구분합니다. 예)
     바닷가 노을 영상 만들어줘 ||| 포근한 카페 감성 ||| 꿈속을 걷는 고래 이야기

3. 프로그램 실행
   - `2_프로그램실행.bat` 을 더블클릭하면 "Sora2 Auto – Aurora Horizon" 창이 열립니다.
   - `🌠 스타트` 버튼을 누르면 프로그램이 전용 Chrome 창을 자동으로 띄우고 DevTools 포트(기본 9222)로 연결합니다.
   - 처음 한 번만 Sora에 로그인하면 해당 프로필(`chrome_profile_plus`)에 저장되어 이후에는 로그인을 반복하지 않아도 됩니다.

4. 자동화 흐름
   - 로그인 후에는 프로그램이 백그라운드에서 Sora Drafts 탭에 직접 프롬프트를 입력하고 제출합니다.
   - 사용자는 다른 브라우저나 다른 Chrome 창에서 자유롭게 인터넷을 이용해도 됩니다.

5. 타이머 조절 & 카운트다운
   - 상단의 `⏱️ 자동 간격` 스핀박스로 30초~3시간 사이 원하는 주기를 설정할 수 있습니다.
   - 값을 바꾸면 바로 저장되고, 현재 타이머가 새 주기로 즉시 업데이트됩니다.
   - 하단 보라색 카운트다운에는 다음 자동 입력까지 남은 시간이 초 단위로 실시간 표시됩니다.

6. 실시간 제어 (버튼 클릭)
   - `⏸️ 일시정지` : 당분간 멈추기
   - `▶️ 다시재생` : 다시 이어서 실행
   - `⚡ 바로실행` : 기다림 없이 즉시 다음 프롬프트 입력
   - `🪽 건너뛰기` : 현재 순서의 프롬프트 제거
   - `🛑 종료` : 전체 작업 중지
   - `🔄 프롬프트 다시불러오기` : prompts.txt 수정 사항을 다시 읽기

7. 자동 저장 & 로그
   - 진행상황은 `sora_state.json` 에 저장되며, 중간에 종료되어도 이어서 재개됩니다.
   - 실행 기록은 `logs` 폴더의 `sora_run_날짜_시간.log` 파일로 남습니다.

8. 문제가 생길 때
   - Chrome이 뜨지 않으면 sora_config.json의 `chrome_executable` 경로를 확인하세요.
   - 로그인 화면에서 멈추면 브라우저에서 직접 로그인 후 `▶️ 다시재생` 또는 `⚡ 바로실행` 버튼을 눌러 주세요.
   - 예상치 못한 오류가 발생하면 logs 폴더의 최신 로그를 첨부해 문의하세요.
