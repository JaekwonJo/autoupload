Sora2 Auto (Codex Edition)

이 폴더에는 감성 UI를 갖춘 Sora Drafts 자동 입력 프로그램이 들어 있습니다.

빠른 시작:
1) 1_처음설치.bat 더블클릭 → 필수 패키지 설치
2) prompts.txt 파일에 프롬프트를 `|||` 로 구분해 적기
3) 2_프로그램실행.bat 더블클릭 → 오로라 창에서 `🌠 스타트`
   - 프로그램이 전용 Chrome 창을 자동으로 띄우고, 한 번 로그인하면 그 프로필에 계속 저장됩니다.
   - 이후에는 백그라운드에서 Sora Drafts를 제어하므로, 다른 브라우저로 자유롭게 서핑해도 됩니다.
   - `⏱️ 자동 간격` 스핀박스로 30초~3시간 사이 원하는 주기를 설정하면, 실시간 카운트다운이 아래에 표시됩니다.

실행 중에는 창의 안내에 따라 Start / Pause / Skip 등을 클릭하세요.
문제가 생기면 logs 폴더의 최신 로그를 확인해 주세요.
