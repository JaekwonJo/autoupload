1. Python 3.11이 설치되어 있다면 `1_처음설치.bat`을 실행해 필수 패키지를 자동으로 설치하세요. (설치가 안 돼 있으면 먼저 python.org에서 3.11을 설치합니다.)
2. `2_프로그램실행.bat`을 더블클릭하면 Sora2 자동 프로그램이 실행됩니다.
3. 프로그램이 뜨면 `📚 프롬프트 세트`를 고르고 `📄 열기`로 텍스트 파일을 편집합니다. 프롬프트는 `|||` 로 구분합니다.
4. `🌠 스타트`를 누르면 전용 Chrome 창이 열리고 Sora Drafts에 프롬프트가 자동 입력/제출됩니다. (최초 1회 로그인 필요할 수 있습니다.)
5. Hailuo 자동(최소 버전): `Hailuo완성.bat`을 더블클릭 → 간격(초) 설정 → `🎯 제출 버튼 지정`, `🧹 초기화 버튼 지정`만 지정해두고 `🌠 시작`을 누르면, [입력 → 제출 → 대기 → 초기화 → 다음 입력] 순서로 반복됩니다.
